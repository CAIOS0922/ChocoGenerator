# KanadeLog
Kanade用のアプリケーションログ
ChocoGeneratorのテスト用Markdown

### OnApplicationStart
アプリケーションが開始された

### OnFirstOpen
初めてKanadeを起動した

### OnMainControllerShow
MainControllerFragmentが全画面表示になった
```
title: String
artist: String
isPlaying: Boolean
```

### OnPlay
楽曲が再生された
```
title: String
artist: String
```
